## Idea

Make a remote control cardboard airplane with ESP32 and NRF24L01

## Materials

```text
Airplane:
    1x ESP32
    1x ESC
    2x 9gr Micro servos
    1x 7.4V 450Mah LiPo Battery
    1x Brushless dc motor
    1x 108mm Propellers
    1x NRF24L01
```

```text
Controller:
    1x ESP32
    1x NRF24L01
    1x 100uF Capacitor
    2x Joystick
```
